Startup parameters

 

 

List of all possible statup parameters:

 

Parameter

Description

Usage

/s + Filename

Scans the given filename

antivirus /s c:\infected.vbs

/gui

Starts the program as usual

antivirus /gui

/update

Lets the program download the latest update

antivirus /update]

/check

Shows the scanfile Dialog

antivirus /check

/t

Shows only the tray windows

antivirus /t
